GameX

GameX é um e-commerce de jogos digitais inspirado na PlayStation Store, criado com HTML, CSS, JavaScript, PHP e MySQL. Oferece uma plataforma interativa para compra de jogos, com carrinho de compras e gerenciamento de usuários e produtos.

CareUp

CareUp é um projeto acadêmico desenvolvido para oferecer soluções acessíveis de seguros de vida e planos de saúde para jovens de 18 a 25 anos. Utilizando HTML, CSS, JavaScript, PHP e MySQL, o site visa tornar esses serviços mais acessíveis para um público jovem.
